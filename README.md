# metagenomic-multi-assembly-wrapper
This repository serves as a multi-engine metagenomic assembly module designed to generate raw assemblies from ONT and Illumina sequencing data. It is a standalone component of a larger assembly comparison pipeline. This wrapper contains Flye, Nextpolish, SPAdes, metaSPAdes, IDBA-UD, MEGAHIT, metaMDBG, and nanoMDBG.
